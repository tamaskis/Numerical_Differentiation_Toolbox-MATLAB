directional2

Directional derivative of a multivariate, scalar-valued function using the backward difference approximation (given two points).

Syntax

Dv = directional2(f,xa,xb,v)

Description

Dv = directional2(f,xa,xb,v) numerically evaluates the directional derivative of at in the direction of using the backward difference approximation.

Example

Approximate the directional derivative of at in the direction of using the directional2 function with the auxiliary point , and compare the result to the true result of .

Approximating the directional derivative,

f = @(x) x(1)^5+sin(x(2))^3;
xb = [5;8];
xa = [5.01;8.01];
v = [10;20];
Dv = directional2(f,xa,xb,v)

Dv =

   3.1366e+04

Calculating the error,

error = Dv-(31250+60*sin(8)^2*cos(8))

  124.9733
