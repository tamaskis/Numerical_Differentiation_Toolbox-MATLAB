fjacobian

Jacobian of a multivariate, vector-valued function using the forward difference approximation.

Example

Approximating the Jacobian,

f = @(x) [x(1);5*x(3);4*x(2)^2-2*x(3);x(3)*sin(x(1))];
x0 = [5;6;7];
J = fjacobian(f,x0)
