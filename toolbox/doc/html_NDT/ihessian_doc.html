ihessian

Hessian of a multivariate, scalar-valued function using the complex-step approximation.

Syntax

H = ihessian(f,x0)
H = ihessian(f,x0,hi,hc)

Description

H = ihessian(f,x0) numerically evaluates the Hessian of with respect to at using a hybrid of complex-step and central difference approximations with default relative step sizes of and , respectively, where is the machine zero.

H = ihessian(f,x0,hi,hc) numerically evaluates the Hessian of with respect to at using a hybrid of complex-step and central difference approximations with user-specified relative step sizes and , respectively.

Example

Approximate the Hessian of at using the ihessian function, and compare the result to the true result of

Approximating the Hessian,

f = @(x) x(1)^5*x(2)+x(1)*sin(x(2))^3;
x0 = [5;8];
H = ihessian(f,x0)

H =

   1.0e+04 *

   2.000000000103592   0.312457274073005
   0.312457274073005  -0.001389784994796

Calculating the error,

error = H-[20*5^3*8,5*5^4+3*sin(8)^2*cos(8);5*5^4+3*sin(8)^2*cos(8),...
    6*5*sin(8)*cos(8)^2-3*5*sin(8)^3]

error =

   1.0e-05 *

   0.103591810329817   0.001091166268452
   0.001091166268452   0.004945257714439
