jacobian2

Jacobian matrix of a multivariate, vector-valued function using the backward difference approximation (given two points).

Syntax

J = jacobian2(f,xa,xb)

Description

J = jacobian2(f,xa,xb) numerically evaluates the Jacobian matrix of at using the backward difference approximation.

Input/Output Parameters

Variable Symbol Description Format
Input f multivariate, vector-valued function () 1×1
function_handle
xa auxiliary point n×1
double
xb point at which to evaluate the Jacobian matrix n×1
double
Output J Jacobian matrix of evaluated at m×n
double

Example

Approximate the Jacobian matrix of

at using the jacobian2 function with the auxiliary point , and compare the result to the true result of

Approximating the Jacobian,

f = @(x) [x(1);5*x(3);4*x(2)^2-2*x(3);x(3)*sin(x(1))];
xb = [5;6;7];
xa = [5.01;6.01;7.01];
J = jacobian2(f,xa,xb)

J =

    1.0000         0         0
         0         0    5.0000
         0   48.0400   -2.0000
    2.0192         0   -0.9589

Calculating the error,

error = J-[1,0,0;0,0,5;0,48,-02;7*cos(5),0,sin(5)]

error =

         0         0         0
         0         0   -0.0000
         0    0.0400   -0.0000
    0.0335         0    0.0000

NOTE: The function and its corresponding Jacobian are from an example on Wikipedia.
