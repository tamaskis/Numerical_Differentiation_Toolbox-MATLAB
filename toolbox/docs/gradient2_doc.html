gradient2

Gradient of a multivariate, scalar-valued function using the backward difference approximation (given two points).

Syntax

g = gradient2(f,xa,xb)

Description

g = gradient2(f,xa,xb) numerically evaluates the gradient of at using the backward difference approximation.

Example

Approximate the gradient of at using the gradient2 function with the auxiliary point , and compare the result to the true result of

Approximating the gradient,

f = @(x) x(1)^5+sin(x(2))^3;
xb = [5;8];
xa = [5.01;8.01];
g = gradient2(f,xa,xb)

   -0.0004

Calculating the error,

error = g-[3125;3*sin(8)^2*cos(8)]
